# Gravus Capital — Internacional

Painel global em tempo real: índices por região (Américas, Europa, Ásia), futuros dos EUA,
curva de Treasuries, câmbio, commodities, cripto e notícias internacionais (PT + EN).

## Rodar localmente

```bash
npm install
npm run dev
```

Acesse: http://localhost:3000

## Deploy no Vercel

1. Suba este projeto no GitHub
2. Acesse https://vercel.com e importe o repositório
3. Deploy — o site abre direto na tela internacional

## Fontes de dados

- Yahoo Finance (índices, futuros, Treasuries, câmbio, commodities, cripto, mega caps)
- InfoMoney Mundo, Bloomberg Línea, Google News (notícias em português)
- CNBC, MarketWatch, Google News US (notícias em inglês)
- TradingView (gráfico S&P 500 e calendário econômico)
